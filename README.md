# Global Population & Demographic Growth Analysis 🌍📈

## Project Overview
This project explores global demographic trends, focusing on the population sizes and growth rates of various countries. The goal of this analysis is to visually identify which regions are experiencing rapid population expansion and which are facing demographic stagnation or decline.

## Tools Used
* **Excel:** Initial data gathering and raw file formatting.
* **Power Query:** ETL (Extract, Transform, Load) processes including data cleaning, handling null values, and ensuring correct data types for macro-economic metrics.
* **Power BI:** Data modeling and interactive dashboard design, utilizing geospatial visuals to map global trends.

## Key Features & Visuals
* **Geospatial Mapping:** A world map visual plotting population sizes to instantly identify high-density regions.
* **Growth Trend Analysis:** Visuals comparing current population sizes against annual growth rates to highlight emerging global markets.
* **Interactive Filtering:** Slicers that allow users to drill down into specific continents, regions, or growth brackets.
